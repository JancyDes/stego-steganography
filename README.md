# Steganography Forensics System

A cybersecurity-based web application to detect hidden data in images and audio using steganography techniques.

## Features

- User Signup & Login
- Encode secret message into image
- Decode hidden message
- Analyze stego risk level
- Risk score classification (Low / Medium / High)
- Modern React UI dashboard

## Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Python Flask
- NumPy
- OpenCV
- Pillow

## How to Run

Backend:

cd stego-backend  
pip install -r requirements.txt  
python app.py  

Frontend:

cd stego-frontend  
npm install  
npm run dev
